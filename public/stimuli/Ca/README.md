# Imágenes de Estímulos para Cancelación (Ca)

Para que la prueba de Cancelación funcione en modo digital, debes subir las imágenes de las láminas aquí.

## Nomenclatura Requerida

El sistema busca las imágenes con el siguiente formato: `item{número}.webp`

*   **Lámina 1 (Ejemplo y Práctica):** `item1.webp` (Contiene ambos ejercicios en la misma hoja)
*   **Lámina 2 (Ensayo 1 - Aleatoria):** `item2.webp`
*   **Lámina 3 (Ensayo 2 - Estructurada):** `item3.webp`

## Formato de Imagen

*   **Formato:** `.webp` (Recomendado por rendimiento) o `.png` / `.jpg` (requiere ajustar el código si cambias la extensión).
*   **Resolución:** Se recomienda al menos 1920x1080px para que se vea bien en tablets.
*   **Orientación:** Horizontal (Landscape).

## Nota

Si no subes las imágenes, el sistema mostrará un mensaje de error indicando que se use el cuadernillo físico.
